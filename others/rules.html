Guidelines

The Members will be divided into two main groups :

  • Tech-hoppers - Those people who are interested in Tech. news, related discussions and occassional presence in meetups.

  • Tech-holics - Those people who are interested in learning and diving into programming and creating some nice apps.

Further Tech-holics will be sub-categorised as follows:

  • Newbies - These members will be working on generating new ideas as well as ideas for existing projects and will be following regular meetups schedule.
    Newbies can work on projects but can not post in Codesters blog.
    ( They need to get their blog post approved by the Dedicated member for posting ).

  • Dedicated - Main role of this group will be blogging, implementation of new ideas and projects, managing the Deadline/Timeline for the projects being taken up and guiding Newbies with their ideas.

All the members will be admitted as Newbie initially and he/she has to complete following criteria(s) to get upgraded to the Dedicated :

  1. At least one of the Newbies' ideas should be implemented.
  2. Minimum attendance ( when all other criterias are fulfilled ) in the meetups should not be less than 75%
  3. He/She should have a minimum of 1000 points on Stackoverflow.

Getting Started with Us

  1. Read all the guidelines before getting started.
  2. You need to create accounts on following sites:
    • Gmail
    • StackOverflow
    • Trello (For Projects Management )
    • Github (For sharing code )
  3. Prefer creating accounts on Trello, Stackoverflow and Github with a common Gmail ID.
